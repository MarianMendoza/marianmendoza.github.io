Projects

CHOMG Best Software Project Award Supported By SIG

During our 3rd year semester, we participated in a competition supported by SIG Susquehanna International Group. I'm proud to receive this award with my fellow team members, Liam Healy, Amy Craven, Luca Gahan and Jack O'Sullivan. CHOMG is our security app that uses motion detection technology, turning webcams that are connected to a Raspberry Pi into advanced security devices. It monitor your space for unexpected movements, notifying you of potential security breaches. Our system process and sends data to a digital ocean cloud server, allowing users to receive real-time alerts through a sleep, user-friendly app. This ensures customers can monitor their property's safety from anywhere, offering peace of mind with ease.

My responsibility to the team was to develop the front end with Java, and .XML. Additionally, I had to adapt to Android Studio to deploy and create the app. I used to Figma to create a friendly UI and achieve a minimalist look. We did version controls with Github, using our own branches and merging at the final stage of development. Click here to view the github repository for CHOMG

NorthWind Table Viewer and Exporter

This was the first project I developed within my internship at the CSO. This application displayed the database in SQL Server Management Studio to a desktop application. This was developed with the framework .NET and I had to gain proficiency in the programming language C#.

This application allows you to view the customers, employees and orders tables within the NorthWind database in SSMS. This allows you to change between table views, search ID and names, filter columns, and change row sizes and font for comforts. Other features include the exporting of your selection in .pdf, and .csv. Many NuGet packages were used to achieve these functionality. Click here to view the github repository for the Northwind Table Viewer and Exporter.
